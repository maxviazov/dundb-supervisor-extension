# D&B כרטיס לסופרוויזר — תוסף דפדפן

תוסף ל-Chrome / Edge שמאפשר להזין **ח.פ.** ולהעתיק **תמונת כרטיס חברה** לשליחה לסופרוויזרים בוואטסאפ.

## איך זה עובד

1. מתחברים פעם אחת ל-[members.dundb.co.il](https://members.dundb.co.il)
2. **משאירים לשונית D&B פתוחה** (אפשר מזעור)
3. מכל דף — פותחים את התוסף, מזינים ח.פ. ולוחצים **חפש**
4. התמונה מועתקת ללוח — **Ctrl+V** בוואטסאפ

התוסף משתמש **בסשן הקיים** בדפדפן (cookies). אין אחסון סיסמאות.

## התקנה (מצב מפתח)

### Chrome
1. `chrome://extensions` → **מצב מפתחים** → **טען פרוסה ללא אצווה**
2. בחרו את התיקייה `dundb-supervisor-extension`

### Edge
1. `edge://extensions` → **מצב מפתח** → **טען לא ארוז**

## כפתורים

| כפתור | מה עושה |
|--------|---------|
| **חפש** | חיפוש לפי ח.פ. + העתקת תמונה ללוח |
| **קח מהדף הנוכחי** | קריאה מכרטיס חברה פתוח ב-D&B |
| **העתק תמונה — הדבק בוואטסאפ** | העתקה חוזרת של PNG ללוח |
| **הורד קובץ PNG** | שמירה מקומית |
| **פתח באתר D&B** | פתיחת כרטיס החברה באתר |

## מה מוצג בכרטיס

- שם, סטטוס, כתובת, טלפונים, אימייל, אתר
- סקור ענפי, ותק עסק, מספר מועסקים
- נתונים עסקיים + סרגל דירוג

## פתרון בעיות

| בעיה | פתרון |
|------|--------|
| לא מחוברים | התחברו ל-D&B והשאירו לשונית פתוחה |
| העתקה נכשלה | לחצו שוב על הכפתור הירוק |
| חסרים שדות | פתחו כרטיס ב-D&B → **קח מהדף הנוכחי** |
